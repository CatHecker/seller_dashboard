README — Панель управления объявлениями
Описание проекта
Проект представляет собой веб‑приложение для управления объявлениями (CRUD + поиск, фильтры, пагинация).
Фронтенд написан на React + TypeScript, Material UI, Vite.
Бэкенд — Fastify, данные хранятся в JSON‑файле.
Интегрирован GigaChat API для автоматической генерации описаний и подсказок по цене.

Требования

- Node.js 18 или выше
- npm
- Git

Установка и запуск

1. Клонирование репозитория
   git clone https://github.com/your-username/seller-dashboard.git

2. Установка зависимостей
   cd seller-dashboard
   npm install
   cd ./server
   npm install
   cd ..

3. Настройка переменных окружения
   cp .env.example .env
   Отредактируйте .env, укажите свои учётные данные GigaChat:
   PORT=8080
   AUTHORIZTION_KEY=ваш_client_secret

4. Запуск в режиме разработки
   Для одновременного запуска клиента и сервера удобно использовать два терминала:

Терминал 1 – сервер:
cd server
npm run dev
Сервер запустится на порту 8080.

Терминал 2 – клиент:
npm run dev
Клиент запустится на порту 5173.
